Json定义

  • 用来保存Js对象的一种方式，和Js对象的写法类似。Json使用文本表示一个Js对象的信息，本质是一个字符串。
  • Js: var obj = { a: "hello", b: "world" }
    Json: ' {"a": "hello", "b": "world"} '

转换方法

  • JSON -> JS: JSON.parse( [json字符串] )
  • JS -> JSON: JSON.stringify( [json对象] )
