如何定义文档内容样式

修改代码格式

  1. 代码书写文档格式在官网的标签插件下可以查看

  2. 1 一行简短代码使用反引号 ``` 加在起始和结束位置

  3. 2 代码块 使用codeblock方式

  4. 进行高亮支持，在themes/主题名/layout/_partial/head文件中
    引入

<link rel="stylesheet" hred="https://oindk07nf.gnssl.com/atom-one-dark.css" media="screen" type="text/css">

博客主题的安装与推荐

Posted on 2021-02-02

  1. 在hexo官网下的Themes部分选择喜欢的目标样式，选择后点击名称，跳转到github。

  2. 根据github中文档中的操作信息进行操作。
    以Cactus为例（ github网址；https://github.com/probberechts/hexo-theme-cactus）
    2.1 命令行，hexo根目录下 ：

     $ git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus

    2.2 clone操作完成后，在根目录下找到config.yml文件，并修改文件内的参数theme：

     vim _config.yml
 找到 theme: yilia
 修改为 theme: cactus

    2.3 然后 hexo s 启动博客来查看

  3. 可以参考github主页上提供的方法来修改博客主题的内容参数。
